Module: backend/Dragon/Card

module:backend/Dragon/Card

This module provides access to Dragon Card NFT minting, manipulation and viewing functionality.

  • Minting new Dragon NFT cards with mint. Requires Web3 instance, DragonMinter smart contract instance and account that will mint. Note that the Web3 instance must "know" the private key for the account. DragonMinter instance can be obtained by calling getDragonMinterInstance.
  • Getting view for a Dragon NFT card with getView by its ID. Requires DragonCardView smart contract instance. The contract instance can be obtained by calling getDragonViewerInstance.
